个人静态博客搭建流程+问题解决方法

Posted on 2020-03-09

内容分为两个部分：

一，个人博客搭建流程步骤
二，过程中遇到的问题及解决办法

框架：Hexo 部署：Github

首先安装nodejs，最好是下载LTS长期版的
https://nodejs.org/en/
打开终端：需切换到root用户 sudo su
依次输入 node -v, npm -v 查看node与npm的版本确保安装成功

由于hexo框架需要nodejs来生成，以上是前置步骤，正常需要依赖npm来安装hexo，但由于国内镜像源安装起来很慢，所以在此之前，先用npm安装一个cnpm
命令：npm install -g cnpm –registry=https://registry.npm.taobao.org

然后利用cnpm来安装hexo
命令：cnpm install -g hexo-cli

建立一个空的文件夹，等下所有关于bolg的东西都是在这个文件夹里生成
接下来生成bolg 命令：sudo hexo init
启动hexo 命令：hexo s

免费部署方式——github

登陆github，新建仓库，仓库命名一定是你的昵称.github.io
接着在文件夹下安装一个git的部署插件
命令：cnpm install –save hexo-deployer-git
需要设置config.yml，使用vim打开

然后就可以部署到远端，命令：hexo d

然后就可以用浏览器访问昵称.github.io这个地址
